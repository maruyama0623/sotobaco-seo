# /analyze-document — 資料解析スキル

## 概要
PowerPoint・PDF・Word・Excel等のドキュメントファイルをMarkItDown-MCPでMarkdownに変換し、内容を分析・要約する。

## 実行手順

### 1. ファイルの確認
- 対象ファイルのパスまたはURLを確認する
- ファイル形式を確認（.pptx / .pdf / .docx / .xlsx 等）

### 2. MarkItDown-MCPで変換
- `mcp__markitdown__convert_to_markdown` を使用してMarkdownに変換する
- ローカルファイルの場合:
  ```
  uri="file:///absolute/path/to/file.pptx"
  ```
- Web上のファイルの場合:
  ```
  uri="https://example.com/document.pdf"
  ```

### 3. 内容の分析
- 変換されたMarkdownを読み込み、構造を把握する
- 社長の指示に応じた分析を行う:
  - **競合調査**: 競合の強み・弱み・差別化ポイントの抽出
  - **営業資料の素材化**: 再利用可能な情報・数値・図表の抽出
  - **ブログ記事の元ネタ**: 記事化できるテーマ・論点の整理
  - **問い合わせ対応**: 資料内の該当情報の特定・回答への活用
  - **その他**: 社長の指示に応じた分析

### 4. 結果の報告
- **要約**: 資料全体の概要（3〜5行）
- **詳細分析**: 社長の指示に応じた分析結果
- **次のアクション提案**: 分析結果をどう活用するかの提案

## 活用場面
- 競合の資料（PDF・スライド）の分析
- 既存の営業資料・提案書の内容再構成
- ホワイトペーパー・技術資料からブログ記事の素材作成
- 顧客から共有された資料の内容把握
- `materials/` フォルダ内の資料解析
