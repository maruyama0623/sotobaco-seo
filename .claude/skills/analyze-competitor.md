# /analyze-competitor — 競合調査スキル

## 概要
参考メディアや競合サービスを調査・分析し、差別化ポイントや記事・施策の改善材料を整理する。

## 実行手順

### 1. 事前準備
- `docs/blog-media.md` の参考メディア一覧を読み込む
- 調査対象のテーマ・キーワードを社長に確認する

### 2. 競合コンテンツ収集
- `docs/blog-media.md` の参考メディア一覧から該当テーマの記事をWebFetchで取得
- 必要に応じてWebSearchで追加の競合記事を探す
- 各記事から以下を抽出:
  - 見出し構成（H2/H3の階層）
  - 扱っている情報・論点
  - ターゲット読者
  - CTA・誘導先
  - 文字数の目安

### 3. 分析・比較
- 競合が共通して扱っているテーマ・情報を整理
- 競合が扱っていない切り口・視点を特定
- 自社サービスならではの強み・解決策を整理
- 情報の深さ・正確さを比較

### 4. レポート作成
以下の形式で社長に報告:
- **調査対象**: テーマ・キーワード・調査した記事一覧
- **競合の共通パターン**: 構成・情報・ターゲットの傾向
- **差別化の余地**: 競合が扱っていない切り口
- **自社への提案**: 記事・施策への具体的な活用案
