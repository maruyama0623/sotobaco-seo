# /analyze-competitor — 競合調査スキル

## 概要
参考メディアや競合サービスを調査・分析し、差別化ポイントや記事・施策の改善材料を整理する。

## 実行手順

### 1. 事前準備
- `docs/blog-media.md` の参考メディア一覧を読み込む
- 調査対象のテーマ・キーワードを社長に確認する

### 2. 競合コンテンツ収集（Web記事）
- `docs/blog-media.md` の参考メディア一覧から該当テーマの記事をWebFetchで取得
- 必要に応じてWebSearchで追加の競合記事を探す
- 各記事から以下を抽出:
  - 見出し構成（H2/H3の階層）
  - 扱っている情報・論点
  - ターゲット読者
  - CTA・誘導先
  - 文字数の目安

### 3. YouTube動画調査
- WebSearchで `site:youtube.com {テーマ・キーワード}` を検索し、関連動画を特定する
- 上位3〜5本の動画からYouTube Transcript MCP（`get_transcript`）で文字起こしを取得する（`lang: "ja"`）
- 各動画から以下を抽出:
  - チャンネル名・再生回数（人気度の指標）
  - 扱っているトピック・解説の切り口
  - 視聴者のニーズ（動画タイトル・冒頭の問題提起から推測）
  - Web記事にはない情報・実演ベースのノウハウ
- **YouTube調査が有効なテーマ**: kintone設定方法、ツール比較、導入事例、カスタマイズ手順
- **YouTube調査が不要なテーマ**: 料金比較、仕様の正確な記述が必要なテーマ

### 4. 分析・比較
- 競合が共通して扱っているテーマ・情報を整理
- 競合が扱っていない切り口・視点を特定（Web記事とYouTube動画の両方から）
- 自社サービスならではの強み・解決策を整理
- 情報の深さ・正確さを比較
- **YouTube独自の発見**: 動画でしか語られていないユーザーの悩み・実践ノウハウを特記する

### 5. レポート作成
以下の形式で社長に報告:
- **調査対象**: テーマ・キーワード・調査した記事/動画一覧
- **競合の共通パターン**: 構成・情報・ターゲットの傾向
- **YouTube動画の傾向**: 人気動画のトピック・ユーザーが求めている情報
- **差別化の余地**: 競合が扱っていない切り口
- **自社への提案**: 記事・施策への具体的な活用案
