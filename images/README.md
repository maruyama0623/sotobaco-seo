# 画像カタログ

ソトバコポータルSEO記事で使用する画像の一覧です。
新しい画像を追加したら、必ずこのファイルに情報を追記してください。

---

## 命名ルール

```
{feature}_{description}.png

例: announce_category.png
```

### feature プレフィックス一覧

| プレフィックス | 対象機能 |
|--------------|---------|
| `announce` | 全体アナウンス機能 |
| `tab` | タブ整理・閲覧制限 |
| `graph` | グラフリアルタイム表示 |
| `app-list` | アプリ一覧埋め込み |
| `notification` | 通知一覧表示 |
| `unprocessed` | 未処理一覧表示 |
| `permission` | 閲覧権限設定（汎用） |
| `space` | スペース単位ポータル |
| `smartphone` | スマートフォン対応 |
| `setup` | 初期設定・OAuth連携手順 |
| `kintone` | kintone標準画面（比較・説明用） |
| `portal` | ポータル全体画面（Before/After等） |

### description の語彙

| description | 意味 |
|------------|------|
| `overview` | 全体像・完成形 |
| `settings` | 設定画面 |
| `before` | 導入前・Before |
| `after` | 導入後・After |
| `category` | カテゴリ分け表示 |
| `list` | 一覧表示状態 |
| `drag-drop` | ドラッグ&ドロップ操作中 |
| `mobile` | スマートフォン表示 |
| `add` | 追加操作画面 |

---

## 画像追加のフロー

```
1. スクリーンショットを撮る
2. 命名ルールに従ってファイル名をつける
3. GitHubのこのリポジトリの images/ フォルダにアップロード
4. Claudeに「画像を追加したのでカタログに追記して」と伝える
5. Claudeが画像を確認してこのREADME.mdに追記する
```

---

## 記事への埋め込み方法

```markdown
![Altテキスト](../images/ファイル名.png)

例:
![ソトバコポータルの全体アナウンス・カテゴリ別表示画面](../images/announce_category.png)
```

**Altテキストは下記カタログの「Altテキスト」列をそのままコピーしてください（SEO効果あり）。**
