# 画像カタログ

ソトバコポータルSEO記事で使用する画像の一覧です。
新しい画像を追加したら、必ずこのファイルに情報を追記してください。

---

## フォルダ構成

```
images/
├── portal/        ← ソトバコポータル製品スクリーンショット（portal_*, tab_*, space_*, space-link_*, graph_*, announce_*, kintone_*, app-list_*, smartphone_*, setup_*, usecase_*, member_*, app-card_*）
├── corporate/     ← コーポレートサイト用画像（hero.*）
├── btone/         ← Btone（Bカート×kintone連携）製品画像（hero_*, about_*, feature_*, step_*, overview_*）
├── logo/          ← 会社ロゴ・シンボル（logo-*, symbol.*）
├── partner/       ← パートナーバッジ（partner_*）
├── icons/         ← アイコン系画像（howto_*）
├── sotobaco_favicon.ico
├── no_image.jpg
└── README.md
```

## 命名ルール

```
{feature}_{description}.png

例: announce_category.png
```

### feature プレフィックス一覧

| プレフィックス | 対象機能 |
|--------------|---------|
| `announce` | 全体アナウンス機能 |
| `tab` | タブ整理・閲覧制限 |
| `graph` | グラフリアルタイム表示 |
| `app-list` | アプリ一覧埋め込み |
| `notification` | 通知一覧表示 |
| `unprocessed` | 未処理一覧表示 |
| `permission` | 閲覧権限設定（汎用） |
| `space` | スペース単位ポータル |
| `space-link` | スペースリンク（アプリカードとしてkintoneスペースを登録する機能） |
| `smartphone` | スマートフォン対応 |
| `setup` | 初期設定・OAuth連携手順 |
| `kintone` | kintone標準画面（比較・説明用） |
| `portal` | ポータル全体画面（Before/After等） |
| `member` | メンバー設定・権限管理 |
| `app-card` | アプリカード固定・表示設定 |
| `usecase` | 活用パターン（LP用スクリーンショット） |
| `partner` | パートナー認定バッジ |

### description の語彙

| description | 意味 |
|------------|------|
| `overview` | 全体像・完成形 |
| `settings` | 設定画面 |
| `before` | 導入前・Before |
| `after` | 導入後・After |
| `category` | カテゴリ分け表示 |
| `list` | 一覧表示状態 |
| `drag-drop` | ドラッグ&ドロップ操作中 |
| `mobile` | スマートフォン表示 |
| `add` | 追加操作画面 |

## 画像追加のフロー

```
1. スクリーンショット・素材を用意する
2. images/ フォルダ直下に置く（サブフォルダへの振り分けはClaudeが行う）
3. Claudeに「画像を追加した」と伝える（他社サイトからの引用の場合は出典URLも伝える）
4. Claudeが画像を確認し、命名規則に従ったリネーム・該当サブフォルダへの移動・README.mdへの追記を行う
5. 他社のサイト・外部素材からの引用画像は、下記「引用画像の出典一覧」にも記載する
```

## 引用画像の出典一覧

**自社で撮影・作成した画像（ソトバコポータルやkintoneを自社環境で操作したスクリーンショット等）は記載不要です。**
他社のWebサイト・ヘルプページ・外部素材から引用した画像のみ、ここに出典を記載してください。

記事内で引用画像を使用する際は、画像の直下に以下の形式で出典を表記してください：

```markdown
出典：[サイト名](URL)
```

| ファイル名 | 出典元 | URL | 備考 |
|-----------|--------|-----|------|
| kintone_portal-default.png | kintone 公式ヘルプ | https://jp.kintone.help/k/ja/portal | kintoneのデフォルトポータル説明ページより引用 |
| kintone_announce-app-grid.webp | saccsy.com | https://saccsy.com/works/baumkuchen-dc/ | kintoneお知らせカスタマイズ事例記事より引用 |
| kintone_apps-overflow.webp | **要確認** | - | 出典元を確認してこの表に追記してください |
| kintone_portal-designer.png | cybozu developer network | https://cybozu.dev/ja/kintone/tips/development/customize/portal/designing/ | Kintone Portal Designerの活用Tipsページより引用 |
| kintone_oshirase-edit.png | kintone 公式ヘルプ | https://jp.cybozu.help/k/ja/portal/manage/how_to_use_announcement.html | お知らせ掲示板の編集ページより引用 |

## 記事への埋め込み方法

```markdown
![Altテキスト](../images/{サブフォルダ}/ファイル名.png)

例:
![ソトバコポータルの全体アナウンス・カテゴリ別表示画面](../images/portal/announce_category.png)
```

**Altテキストは下記カタログの「Altテキスト」列をそのままコピーしてください（SEO効果あり）。**
